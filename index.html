คำถามที่พบบ่อย

คุณมีคำถาม เรา มีคำตอบ!

Q1. เว็บไซต์จ่ายเป็นรายปีหรือจ่ายครั้งเดียวจบ?

ค่าใช้จ่ายของเว็บไซต์ของเราจะแบ่งเป็น 2 ส่วน คือ
– ค่าทำเว็บไซต์ในส่วนนี้จะเป็นการจ่ายครั้งเดียวจบ
– ค่าโฮสติ้งเเละค่าดูเเลเว็บไซต์ คิดเป็นรายปี เริ่มนับตั้งแต่เว็บไซต์เริ่มใช้งานจริง

Q2. แก้ไขเว็บไซต์ได้กี่รอบ?

เพื่อรักษามาตรฐานการส่งมอบเว็บไซต์ และได้ตามระยะเวลาที่ตกลงทางเราขอกำหนดจำนวนครั้งการปรับแก้เว็บไซต์สูงสุดเพียง 3 รอบครับ

Q3. ใช้อะไรในการเขียนเว็บไซต์?

เราใช้ภาษา HTML , CSS , JS , PHP , MYSQL เเละอื่นๆในการเขียนเว็บไซต์ครับ ^^

Q4. ถ้าทำเว็บไซต์ใหม่แต่อยากใช้ชื่อเว็บไซต์เดิมหรือ URL เดิมต้องทำยังไง?

เมื่อลูกค้าใช้บริการทำเว็บไซต์กับเราเราจะมีทีมงานช่วยดำเนินการแจ้งย้ายชื่อเว็บไซต์ (Domain Name) ให้ เพียงแค่แจ้งข้อมูลที่เคยจดทะเบียนซื้อ เพื่อให้ทีมงานของเราช่วยประสานงานให้ครับ

Q5. จ้างทำเว็บการพนันได้หรือไม่?

คำตอบคือไม่ได้ครับ ทางเราไม่ได้รับเขียนเว็บที่ผิดกฏหมายครับ

Q6. Domain Name คืออะไร ?

Domain name (โดเมน เนม) คือ ชื่อที่ถูกเรียกแทนการเรียกเป็นหมายเลขอินเทอร์เน็ต (IP Address) เนื่องจากการจดจำหมายเลข IP ถึง 16 หลัก ทำให้ยุ่งยาก และไม่สามารถจำได้เวลาท่อง เที่ยวไปในระบบอินเทอร์เน็ต จึงนำชื่อที่เป็นตัวอักษรมาใช้แทน ซึ่งมักจะเป็นชื่อที่สื่อความหมายถึง หน่วยงาน หรือเจ้าของเว็บไซต์นั้นๆ เช่น ชื่อโดเมนของเราคือ ThaiHostClub และคุณสามารถเข้าใช้งาน เวบไซต์ของเราผ่าน www.wemakeweb.in.th เป็นต้น

Q7. Hosting ของ wemakeweb.in.th รองรับ SSL มั้ย?

รองรับครับ ถ้าใช้บริการเขียนเว็บกับทางเรา เรามีบริการทำ SSL CERTIFICATE ให้ฟรีไม่คิดค่าใช้จ่ายเลยครับ ^^

Q8. มีการจำกัดพื้นที่ใช้งานบนเว็บไซต์หรือไม่?

ปัจจุบัน wemakeweb.in.th ยังไม่มีการจำกัดพื้นที่ใช้งานบนเว็บไซต์ ทั้งนี้ wemakeweb.in.th ขอสงวนสิทธิ์เปลี่ยนแปลงเงื่อนไขโดยไม่จำเป็นต้องแจ้งให้ทราบล่วงหน้า

Q9. SEO คืออะไร มีประโยชน์อย่างไร?

Search Engine Optimization หรือ SEO เป็นการจัดการเนื้อหาภายในเว็บไซต์ โดยการใช้คำค้น (Keyword) เพื่อให้ติดอันดับการค้นหาใน Google ในตำแหน่งที่ดีที่สุด หลายธุรกิจจึงให้ความสำคัญกับการทำ SEO เพราะเป็นอีกหนึ่งวิธีในการทำการตลาดที่เห็นผลอย่างยั่งยืน ช่วยเพิ่มโอกาสให้ลูกค้าค้นพบกับสินค้าหรือบริการในหน้าแรกของ Google โดยไม่เสียค่าใช้จ่าย หรือค่าโฆษณาแม้แต่บาทเดียว
